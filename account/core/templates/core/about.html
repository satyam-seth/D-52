How to use?

We are here to help you.

  1. Go to Add Data page from the navigation bar. It will show there if you are already logged in.
  2. If you want to add a new record for the item, then follow the following steps and fill the Add Item Record form-
    • Select Date.
    • Select your name.
    • Enter item name.
    • Enter item price.
    • Click Add button.
    • You will get a message that Your item record successfully added.
  3. If you want to add a new record for the water, then follow the following steps and fill the Add Water Record form-
    • Select Date.
    • Select quantity.
    • Click Add button.
    • You will get a message that Water record successfully added.
  1. If you want to see all items records, then go to See Records page from the navigation bar.
  2. If you want to see any person's personal items records, then press the Detailed View button which is on the home page.
  3. If you want to see water records, then press the Detailed View button which is on the home page.
  4. Apart from this, if you want, you can write the name of an item in the Search Bar and see all the records related to that item.
If you want to see all progress reports, then go to Progress Reports page from the navigation bar.
If you want to download the excel file of records, then go to Download Excel Sheet page in More section of the navigation bar and press Download button of that record.
Go to Feedback page in More section from the navigation bar then follow the following steps and fill the Suggestion form-
  • Enter your name.
  • Enter problom topic or suggestion topic.
  • Enter problom message or suggestion message.
  • Click Submit button.
